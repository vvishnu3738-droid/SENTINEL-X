# SENTINEL-X Feature List

## Core Features

### 1. Autonomous Patrol
The robot autonomously patrols predefined industrial areas without continuous human control.

### 2. Environmental Monitoring
Continuously monitors:
- Temperature
- Humidity
- Air Quality
- Gas Leakage
- Smoke
- Distance to Obstacles
- Battery Voltage

### 3. AI Risk Assessment
Uses embedded AI to classify the environment as:
- Safe
- Warning
- Critical

### 4. Worker Alert System
Provides:
- OLED Status
- RGB LED Warning
- Buzzer Alert
- Dashboard Notification

### 5. Live Data Monitoring
Transmits sensor data to a web dashboard over Wi-Fi.

### 6. Battery Monitoring
Continuously checks battery level and estimates remaining operation time.

### 7. Automatic Dock Charging
When the battery reaches the defined threshold, the robot automatically returns to the charging dock using magnetic pogo-pin charging.

### 8. Resume Patrol
After charging, the robot resumes patrol from the previous mission.

### 9. Event Logging
Stores safety events and patrol history.

### 10. Future Expansion
- Thermal Camera
- AI Vision
- Digital Twin
- Multi-Robot Coordination
