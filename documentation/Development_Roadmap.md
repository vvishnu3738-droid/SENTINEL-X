# SENTINEL-X Development Roadmap

## Project Duration

Total Duration: 28 Days

---

# Phase 1 – Planning & Design (Day 1–5)

Objectives:

- Finalize project architecture
- Freeze features
- Complete documentation
- Finalize Bill of Materials (BOM)
- Design robot body
- Design charging dock
- Prepare GitHub repository

Deliverables:

- Project documentation
- CAD sketches
- Component list
- Architecture diagrams

---

# Phase 2 – Hardware Integration (Day 6–15)

Objectives:

- Assemble robot chassis
- Install Arduino UNO Q
- Connect sensors
- Connect motor driver
- Build charging dock
- Test power system

Deliverables:

- Robot hardware ready
- Sensors working
- Charging dock prototype

---

# Phase 3 – Firmware Development (Day 16–22)

Objectives:

- Sensor reading
- Motor control
- Obstacle avoidance
- Battery monitoring
- AI decision logic
- Dashboard communication

Deliverables:

- Robot firmware
- AI logic
- Dashboard connectivity

---

# Phase 4 – Testing & Optimization (Day 23–26)

Objectives:

- Sensor calibration
- Navigation testing
- Charging dock testing
- Battery endurance testing
- Bug fixing

Deliverables:

- Stable robot
- Test reports

---

# Phase 5 – Final Submission (Day 27–28)

Objectives:

- Demo video
- GitHub update
- Project report
- PPT
- Final testing

Deliverables:

- Competition-ready project
