# SENTINEL-X REGULATED POWER NETWORK BLUEPRINT

1. PRIMARY ENERGY STORAGE CELL
   - Power Unit: 12.8V LiFePO4 Battery Pack (6Ah Nominal Capacity).
   - Guard System: Integrated 15A Battery Management System (BMS) Protection PCB.

2. HIGH-CURRENT INDUCTIVE POWER SYSTEM RAIL (Noisy Actuator Domain)
   - Route: Raw 12.8V Battery Output is hardwired directly to the L298N Motor Driver Input Terminal Blocks.
   - Load: Powers 4x 12V High-Torque Side-Shaft DC Geared Motors.
   - Engineering Protocol: Isolates back-EMF spikes generated by electrical motors, preventing system crashes or micro-controller brown-outs.

3. LOW-CURRENT STEADY LOGIC SYSTEM RAIL (Clean Processing Domain)
   - Route: Raw 12.8V Battery Output feeds directly into an LM2596 DC-DC Buck Converter.
   - Buck Tuning: Adjusted via onboard potentiometer to output a steady 5.0V DC.
   - 5.0V Rail Distribution Net:
     * Arduino UNO R4 '5V' Pin (Bypasses linear regulator to completely avoid thermal heating).
     * Sensor VCC Lines: DHT22, MPU6050, BH1750.
     * MQ-135 & MQ-2 Internal Heater Elements (Draws high structural current: ~150mA each).
     * DFPlayer Mini Processing Chip & 8-ohm HMI Output Speaker.
