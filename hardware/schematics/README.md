# SENTINEL-X

## AI Autonomous Industrial Safety Patrol Robot

---

## Project Overview

SENTINEL-X is an AI-powered autonomous industrial safety patrol robot developed for continuous 24/7 industrial monitoring. The robot autonomously patrols hazardous environments, collects real-time sensor data, performs AI-based risk analysis, alerts workers during emergencies, and automatically returns to its charging dock when the battery is low before resuming its patrol.

---

## Objectives

- Enhance industrial worker safety.
- Perform autonomous industrial patrol.
- Monitor environmental conditions in real time.
- Detect hazards using AI.
- Provide intelligent alerts.
- Enable continuous 24/7 monitoring.
