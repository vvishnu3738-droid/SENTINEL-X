1. PRIMARY POWER INTAKE INTERFACE (12.8V HIGH-CURRENT RAIL)
--------------------------------------------------------------------------------
[LiFePO4 Battery (+) Terminal] ------> [10A Safety Switch] ------> [L298N VCC Terminal]
                                                           └-----> [LM2596 INPUT (+) Node]

[LiFePO4 Battery (-) Terminal] ------> [Main Common Ground Node] ---> [L298N GND Terminal]
                                                           └-----> [LM2596 INPUT (-) Node]

2. STEP-DOWN REGULATION LAYER (5.0V CLEAN POWER RAIL)
--------------------------------------------------------------------------------
[LM2596 OUTPUT (+) Node] ------------> [5V Clean Power Distribution Rail] ---> [Arduino 5V Pin]
                                                                   ├-----> [MQ-2 VCC Pin]
                                                                   ├-----> [MQ-135 VCC Pin]
                                                                   ├-----> [DHT22 VCC Pin]
                                                                   ├-----> [MPU6050 VCC Pin]
                                                                   └-----> [DFPlayer Mini VCC Pin]

[LM2596 OUTPUT (-) Node] ------------> [Main Common Ground Node] -------> [Arduino GND Pin]
                                                                   ├-----> [All Sensor GND Pins]
                                                                   └-----> [DFPlayer Mini GND Pin]

3. MICROCONTROLLER DIGITAL / ANALOG SIGNAL PROCESSING TRACKS
--------------------------------------------------------------------------------
[Motion Control & Motor Driver Interface]:
- Arduino Pin [D3]  (PWM Output) ----------> L298N Speed Pin [ENA] (Left Motor Speed)
- Arduino Pin [D4]  (Digital Output) ------> L298N Direction Pin [IN1] (Left Motor Direction)
- Arduino Pin [D5]  (PWM Output) ----------> L298N Speed Pin [ENB] (Right Motor Speed)
- Arduino Pin [D6]  (Digital Output) ------> L298N Direction Pin [IN2] (Right Motor Direction)

[Obstacle Detection & Environmental Sensors]:
- Arduino Pin [D8]  (Digital Input)  <------ HC-SR04 Ultrasonic Sensor [Echo Pin]
- Arduino Pin [D9]  (Digital Output) ------> HC-SR04 Ultrasonic Sensor [Trigger Pin]
- Arduino Pin [D13] (Digital Input)  <------ DHT22 Temperature & Humidity Sensor [Data Output Pin]
- Arduino Pin [A0]  (Analog Input)   <------ MQ-135 Air Quality Sensor [Analog Output Pin]
- Arduino Pin [A1]  (Analog Input)   <------ MQ-2 Smoke/Gas Sensor [Analog Output Pin]

[High-Speed Communication & Serial Data Buses]:
- Arduino Pin [A4] (I2C SDA Bus) <---------> MPU6050 [SDA Pin] & BH1750 [SDA Pin] (Shared Bus)
- Arduino Pin [A5] (I2C SCL Bus) ---------> MPU6050 [SCL Pin] & BH1750 [SCL Pin] (Shared Bus)
- Arduino Pin [D2] (Software RX) <--------- DFPlayer Mini MP3 Module [TX Pin]
- Arduino Pin [D7] (Software TX) ---------> DFPlayer Mini [RX Pin via 1KΩ Resistor]

[Alarms & Human-Machine Interface (HMI)]:
- Arduino Pin [D10] (Digital Output) ------> Common Cathode RGB LED [Red Anode]
- Arduino Pin [D11] (Digital Output) ------> Common Cathode RGB LED [Green Anode]
- Arduino Pin [D12] (Digital Output) ------> 5V Emergency Buzzer (+)

================================================================================
ENGINEERING NOTE:
To suppress serial communication noise and protect the communication line,
a 1KΩ resistor must be connected in series with the DFPlayer Mini RX wire.
